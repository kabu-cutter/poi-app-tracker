# poi-app-tracker / アプリ進行中

ポイ活ゲームアプリ案件の進行状況を記録・管理するChrome拡張です。
Chromeウェブストアでの表示名は **アプリ進行中** を想定しています。

## 主な機能

- 現在開いている案件ページのタイトルとURLを1クリックで記録
- 複数案件の一覧管理
- サイト名、報酬、インストール日、開始日、期限、状態、条件、メモ、進行ログの保存
- 期限が近い案件の表示強化
- ステータス管理
  - 未インストール
  - 進行中
  - 達成
  - 承認待ち
  - 却下
  - 問い合わせ中
  - 撤退
  - 保留
- 日付つき進行ログ
- 一覧での最新ログ表示
- CSV出力
- JSONバックアップ・復元
- データは `chrome.storage.local` に保存
- 外部サーバー送信なし

## インストール方法（開発用）

1. このリポジトリをダウンロードまたは clone
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」を押す
5. このリポジトリのフォルダを選択

## データ保存について

保存キーは以下です。

```text
appProgressItems
```

v0.5以前のデータも、同じ拡張IDで更新する場合は継続利用できます。v0.6では既存データに `logs` 配列を追加しますが、未入力なら空配列として扱います。
別フォルダとして読み込むとChrome上では別拡張扱いになり、ローカルデータが分かれる場合があります。

移行やバックアップには、案件一覧画面の **バックアップJSON出力** と **JSON復元** を使ってください。

## アイコンについて

この拡張では、`manifest.json` に `icons` / `default_icon` を指定していません。
Chromeが拡張名から自動生成する簡易アイコン、つまり「ア」表示を使う方針です。
